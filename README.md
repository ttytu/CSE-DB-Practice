## CSE3207 Database Project

### Project Structure

- GitHub 
- Docker 
- DB: MySQL `/mysql`
- API: FastAPI Python `/api`
- Frontend: Reaact JS `/frontend` 

--- 

### GitHub

```shell
git clone https://github.com/ttytu/CSE3207-Database.git

### Dev Env - Docker Compose

```
